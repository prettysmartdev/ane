# Setup

## User installation

Download:
- Pre-built static binaries from GitHub Releases (linux-x86_64, linux-aarch64, macos-x86_64, macos-aarch64)
- `cargo install ane` from crates.io
- Build from source: `cargo build --release`

Initial configuration:
- No configuration required — ane works out of the box with sensible defaults
- Optional: create `$HOME/.ane/config.toml` to customize keybindings and theme

Superuser access:
- Not required — ane runs entirely in userspace
